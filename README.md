# Electricity Consumption Analysis in India

## Project Overview
This project analyzes electricity consumption patterns across Indian states and regions using Tableau.

The dataset includes electricity usage data for 2019 and 2020, allowing exploration of trends before and during the COVID-19 pandemic.

## Tools Used
- Tableau Public
- Data Visualization
- Data Analysis

## Key Visualizations
- Monthly Electricity Consumption Trend
- Regional Electricity Usage Comparison
- State-wise Electricity Consumption Map
- Top Electricity Consuming States

## Dashboard & Story
An interactive Tableau dashboard and story were created to present insights about electricity consumption trends across India.
